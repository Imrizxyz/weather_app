# Weather App 🌤

This project fetches real-time weather data using OpenWeather API.

## Features
- Fetch weather using API
- Multi-city support
- Smart temperature logic (Hot / Cold / Normal)

## Versions

### V1 (Basic)
- Simple API call
- Displays temperature

### V2 (Improved)
- Multi-city input
- Smart logic added
- Cleaner output

## Tech Used
- Python
- Requests
- OpenWeather API

## How to Run
1. Install requests:
   pip install requests

2. Run:
   python weather_v2.py

3. Enter city name

## Note
Replace YOUR_API_KEY with your OpenWeather API key.
